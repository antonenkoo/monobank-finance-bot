# Monobank Finance Bot

Личный Telegram-бот для автоматического учёта финансов.

- Получает транзакции с карты Monobank в реальном времени (через webhook)
- Сохраняет все транзакции в базу данных Notion с категориями
- Позволяет вручную добавлять наличные расходы прямо из Telegram
- Поддерживает шаблоны для повторяющихся трат
- Управление полностью через кнопки в Telegram

---

## Что тебе понадобится

Перед установкой убедись, что у тебя есть:

- **Python 3.10+** — [скачать на python.org](https://www.python.org/downloads/)
- **Git Bash** (рекомендуется для Windows) — [скачать тут](https://git-scm.com/download/win)
- **Telegram-аккаунт** и бот, созданный через [@BotFather](https://t.me/BotFather)
- **Аккаунт Monobank** с личным токеном
- **Аккаунт Notion** с двумя базами данных (транзакции + категории)
- **Аккаунт ngrok** (бесплатный) — для получения webhook от Monobank

## Ручная установка (шаг за шагом)

### Шаг 1 — Проверь версию Python

Открой Git Bash и введи:

```bash
python --version
```

Должно быть `Python 3.10` или выше. Если меньше — скачай актуальную версию с [python.org](https://www.python.org/downloads/).

---

### Шаг 2 — Создай виртуальное окружение

В папке проекта:

```bash
python -m venv venv
source venv/Scripts/activate
```

После активации в начале строки появится `(venv)` — это значит всё правильно.

> **Важно:** Каждый раз при открытии нового терминала нужно снова активировать venv командой `source venv/Scripts/activate`

---

### Шаг 3 — Установи зависимости

```bash
pip install -r requirements.txt
```

Подожди, пока все пакеты установятся (обычно 1–2 минуты).

---

### Шаг 4 — Создай файл конфигурации

```bash
cp .env.example .env
```

---

### Шаг 5 — Получи Telegram Bot Token

1. Открой Telegram и найди [@BotFather](https://t.me/BotFather)
2. Напиши `/newbot`
3. Придумай имя бота (например: `My Finance Bot`)
4. Придумай username (должен заканчиваться на `bot`, например: `myfinance_bot`)
5. BotFather отправит тебе токен вида `7123456789:AABBCCddEEFFggHHiiJJkkLLmmNN`

Открой файл `.env` в любом текстовом редакторе и вставь токен:

```
TELEGRAM_BOT_TOKEN=7123456789:AABBCCddEEFFggHHiiJJkkLLmmNN
```

---

### Шаг 6 — Запусти бота

```bash
python main.py
```

Ты увидишь:

```
──────────────────────────────────────────────────
  Monobank Finance Bot запущен
  Статус: ⚠️  требует настройки (/config)
──────────────────────────────────────────────────
```

Не закрывай терминал — бот работает, пока он открыт.

---

### Шаг 7 — Первый запуск в Telegram

1. Открой своего бота в Telegram
2. Нажми **Start** или напиши `/start`
3. Бот покажет главное меню с кнопками внизу экрана
4. Нажми **⚙️ Настройки** → **⚙️ Конфигурация** — и заполни все поля

## Настройка всех токенов (внутри бота)

Все настройки, кроме `TELEGRAM_BOT_TOKEN`, делаются прямо в боте через меню.

### Monobank токен

1. Открой [monobank.ua](https://www.monobank.ua/) в браузере
2. Войди в личный кабинет → раздел **Особисті дані**
3. Скопируй **Токен** (или нажми «Отримати токен»)
4. В боте: **⚙️ Настройки → ⚙️ Конфигурация → Monobank Personal Token**
5. Вставь токен и отправь

### Monobank Account ID

После того как задал Monobank токен:
1. В боте: **⚙️ Настройки → ⚙️ Конфигурация → 📋 Выбрать аккаунт Monobank**
2. Бот загрузит список твоих карт
3. Нажми на нужную карту — ID сохранится автоматически

### Notion API Key

1. Открой [notion.so/my-integrations](https://www.notion.so/my-integrations)
2. Нажми **+ New integration**
3. Придумай название (например: `Finance Bot`)
4. Нажми **Submit**
5. Скопируй **Internal Integration Secret** (начинается с `secret_...`)
6. В боте: **⚙️ Настройки → ⚙️ Конфигурация → Notion API Key**

### Настройка баз данных Notion

Тебе нужны две базы данных в Notion.

**База транзакций** — создай базу данных со свойствами:

| Свойство | Тип |
|----------|-----|
| Name | Title (создан по умолчанию) |
| Date | Date |
| Amount | Number |
| Notes | Rich text |
| Categories | Relation → база категорий |

**База категорий** — создай базу данных только с полем:

| Свойство | Тип |
|----------|-----|
| Name | Title |

Добавь в категории нужные тебе записи: `Продукты`, `Транспорт`, `Кафе` и т.д.

**Подключи интеграцию к обеим базам:**
1. Открой базу данных в Notion
2. Нажми `···` (три точки) в правом верхнем углу
3. Выбери **Connections** → найди свою интеграцию → нажми **Confirm**
4. Повтори для второй базы

**Получи ID баз данных:**

Открой базу в браузере. URL выглядит так:
```
https://www.notion.so/workspace/название-базы-abc123def456...
```

ID базы — это 32 символа (буквы и цифры) после последнего `/` и до `?`.  
Пример: `abc123def456789012345678901234ab`

### Ngrok Auth Token

Ngrok создаёт публичный URL для webhook Monobank (без него транзакции не будут приходить автоматически).

1. Зарегистрируйся на [ngrok.com](https://dashboard.ngrok.com/signup) (бесплатно)
2. Войди в личный кабинет → раздел **Your Authtoken**
3. Скопируй токен
4. В боте: **⚙️ Настройки → ⚙️ Конфигурация → Ngrok Auth Token**

> После сохранения токена ngrok нужно перезапустить бота (`Ctrl+C` → `python main.py`)

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/config` | Открыть настройки |
| `/add` | Добавить ручную транзакцию |
| `/create_template` | Создать шаблон транзакции |
| `/debug` | Вкл/выкл режим отладки |
| `/cancel` | Отменить текущее действие |

**Кнопки главного меню:**
- **➕ Добавить** — ручное добавление транзакции (наличные и т.д.)
- **📋 Шаблоны** — использовать, создавать, редактировать шаблоны
- **⚙️ Настройки** — изменить конфигурацию

## Режимы работы

### Тихий режим (по умолчанию)
- Транзакции от Monobank записываются в Notion автоматически и без уведомлений
- Доступна команда `/add` для ручного добавления

### Режим отладки (`/debug`)
- Каждая транзакция от Monobank → сообщение в Telegram с подробностями
- `/add` недоступен в этом режиме

## Troubleshooting

| Проблема | Решение |
|----------|---------|
| `ModuleNotFoundError: No module named 'telegram'` | Активируй venv: `source venv/Scripts/activate` |
| `TELEGRAM_BOT_TOKEN не задан` | Открой `.env` и добавь токен |
| Бот не отвечает в Telegram | Проверь, что `python main.py` запущен в терминале |
| Webhook не приходит от Monobank | Проверь NGROK_AUTH_TOKEN, перезапусти бота |
| Кнопка «Выбрать аккаунт» не показывает карты | API Monobank: запрос можно делать не чаще раза в 60 секунд. Подожди минуту и попробуй снова |
| Ошибка при сохранении в Notion | Убедись, что интеграция подключена к обеим базам данных |
| `RuntimeError: event loop` | Используй Python 3.10+ |
| После перезапуска бот не помнит настройки | Настройки хранятся в `.env` — файл не был удалён? |
